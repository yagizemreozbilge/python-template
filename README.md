## Project Description

Python Template is a basic project template designed to help you get started with your Python projects. It provides a modular structure, essential configuration files, and a foundational project layout. This template is ideal for developers looking to quickly set up a new project.

## Features

- Modular project structure
- Basic Python coding standards
- Documentation support with Sphinx, pdoc, and MkDocs
- Testing support with `pytest`
- Multi-Python version testing with `tox`
